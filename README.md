# Mind Bridge: Student Mental Health & Support Platform

Mind Bridge is a web-based platform designed to help students at North South University access mental health support, connect with peer supporters, and find helpful resources. The goal is to make it easier for students to get help, track their wellbeing, and reach out in times of crisis.

## Features
- Peer Support Chat: Talk to trained peer supporters in a safe, confidential space.
- Crisis Resources: Quick access to emergency hotlines and support information.
- Appointment Scheduling: Book sessions with counselors or peer supporters.
- Progress Tracking: Visualize your emotional state and therapy progress over time.
- Resource Library: Articles, guides, and exercises for mental health and wellbeing.

## How to Use
1. Login/Register: Sign up with your NSU email or log in to your account.
2. Dashboard: View your progress, upcoming sessions, and quick links to all features.
3. Chat: Start a conversation with a peer supporter or submit an anonymous report.
4. Schedule: Book appointments with available counselors or peer supporters.
5. Resources: Browse helpful articles, guides, and exercises.
6. Crisis: Access emergency contacts and support if you need urgent help.

## Setup Instructions
1. Clone or download this repository.
2. Open the project folder in your code editor.
3. All code is in HTML, CSS, and JavaScript (no backend required for demo).
4. To set up the database, use the `mind_bridge.sql` file in your MySQL server.
5. Open `index.html` in your browser to start using the platform.

## Database
- The `mind_bridge.sql` file contains all the tables and sample data for the platform.
- Import it into your MySQL server to enable data features.
